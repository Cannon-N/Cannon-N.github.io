Freshman App Academy

This project was an assignment I completed my freshman year of high school. Each person in the class was tasked to make a game using Unity. My first attempt resulted in a poorly working pacman. However my second time making a game resulted in a endless platform runner shown by the screenshot below. The name I chose for the game was The End Runner, kind of has a creepy ring to it, probably sounded cooler as a freshman. The coding of this project was done in c# which was hard to learn, and the remainder was done with unity extensions such as managing screens and using the scenes to make the runner.

In this screenshot you can see the player on the left-side of the screen, partially cut off. You control the player only by pressing the spacebar to jump from platform to platform. The goal of the game is to last as long as possible by jumping to each consecutive platform. This project was fun because I have been interested in endless runners for a good while and seeing how to make one was an interesting experience. The best part of the project was completing the project.
